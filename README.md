🚨 Disaster Relief and Management System (DRMS)

The Disaster Relief and Management System (DRMS) is a C++-based Data Structures and Algorithms (DSA) project designed to simulate real-world emergency response operations. The system models cities as a graph network and efficiently manages disaster situations using multiple core DSA concepts including graphs, trees, heaps, queues, stacks, linked lists, and sorting algorithms.

This project focuses on optimizing disaster response by finding the shortest rescue routes, prioritizing emergencies based on severity, and maintaining structured records of all operations.

🧠 Key Idea

The system acts as a simplified emergency response engine where:

Cities are represented as graph nodes
Roads are weighted edges (distance/time)
Emergencies are processed based on priority
Rescue operations are optimized using graph algorithms

⚙️ Features
🌐 Graph representation of cities and road networks
🚑 Emergency prioritization using Priority Queue (Heap)
🛣️ Shortest path calculation using Dijkstra’s Algorithm
🔄 BFS and DFS traversal for network exploration
🌳 Disaster record management using Binary Search Tree (BST)
📜 System logging using Linked List
📊 Sorting of rescue teams based on distance
📦 Efficient data handling using arrays and STL structures

🧩 Data Structures & Algorithms Used
Graph (Adjacency List)
Priority Queue (Heap)
Queue (BFS traversal)
Stack (DFS recursion concept)
Binary Search Tree (BST)
Linked List
Sorting Algorithms (STL sort / custom logic)
Dijkstra’s Algorithm
BFS & DFS

🎯 Objective

The main objective of this project is to demonstrate the practical application of fundamental Data Structures and Algorithms in solving real-world problems such as disaster management, emergency response optimization, and resource allocation.

🖥️ Tech Stack

Language: C++
Concepts: OOP + DSA
Platform: Console-Based Application

🚀 Outcome

This project strengthens understanding of core DSA concepts while simulating a real-world system used for emergency management and disaster relief planning. It is also suitable for academic submission and portfolio showcasing.
